Discrub Export

Format: HTML (Discord Layout) · Generated by Discrub

Getting Started

  • Open shell.html — This is the main entry point. It provides a Discord-like interface with a server sidebar, channel list, and top bar.
  • Navigate channels — Click channels in the left sidebar to switch between them.
  • Toggle theme — Click the moon/sun icon (☾/☀) in the top bar to switch between dark and light modes. Your preference is saved.

Features

  • Search — Use the search bar to find messages by content. Matches are highlighted in yellow.
  • Pagination — Large exports are split into pages. Use the navigation buttons at the bottom to move between pages.
  • Media — If media was included, avatars, attachments, and emojis are in the avatars/, media/, and emojis/ folders. Images can be clicked to open a full-size lightbox.
  • User profiles — Click any avatar or username to see user details, badges, and message count.
  • Reactions — Click reaction counts to see who reacted (if detailed reaction data was exported).
  • Threads — Thread messages may be in separate files inside a threads/ folder.

File Structure

  • shell.html — Main entry point (Discord layout)
  • *.html / *.csv / *.json — Exported message data
  • avatars/ — Downloaded user avatars
  • media/ — Downloaded attachments (images, videos, audio)
  • emojis/ — Downloaded custom emojis
  • threads/ — Thread message files (if threads were exported separately)
